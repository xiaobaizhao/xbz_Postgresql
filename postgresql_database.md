## 创建数据库(create database)
postgreSQL 创建数据库有三种方式，分别为：
> 1. 使用 **create database** SQL语句来创建  
> 2. 使用 **createdb** 命令来创建
> 3. 使用 **pgAdmin** 工具
   
因精力有限，目前只研究**create database** 方式创建数据库。 

#### 数据库创建语法如下：
> CREATE DATABASE dbname;  
>> dbname：自定义数据库的名称
