## postgreSQL 快捷键
- \l ： 用于查看已经存在的数据库
- \c dbname : 进入对应的数据库(dbname)中
- \d ：用于查看数据表是否存在
- \d tablename ：用于查看数据表的表结构信息
- \q ：退出当前数据命令窗口
- \di: 列出数据库的所有索引
